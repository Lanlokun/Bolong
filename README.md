# Bolong

Bolong is a cross-border ecommerce platform connecting the Chinese market to consumers in The Gambia, with integrated logistics and order tracking.

## Features

- Product marketplace
- Cart and checkout system
- Admin dashboard
- Order tracking with delivery timeline
- Role-based authentication
- Search and filtering

## Tech Stack

Frontend:
- Next.js
- Tailwind CSS

Backend:
- Node.js
- Express
- PostgreSQL
- Sequelize

## Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev

## Environment Variables

Create `.env` files in both client and server.
